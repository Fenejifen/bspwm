## 快捷键大全

`printscreen` 截图

`super+w` 浏览器

`super + shift + enter` 终端

`super + e` 文件管理器

`super + shift + d` 应用启动器

`super + shift + s` 重新载入快捷键设置

`super + hjkl` 切换注视窗口

`ctrl + shift + esc` 任务管理器

`alt + F2`

`super + x` 退出管理界面

`super + esc` 鼠标选择关闭窗口
